1    Translations
2    ============
3    
4    WTForms uses gettext to provide translations. Translations for various
5    strings rendered by WTForms are created and updated by the community. If
6    you notice that your locale is missing, or find a translation error,
7    please submit a fix.
8    
9    
10   Create
11   ------
12   
13   To create a translation, initialize a catalog in the new locale:
14   
15   ```
16   $ python setup.py init_catalog --locale <your locale>
17   ```
18   
19   This will create some folders under the locale name and copy the
20   template.
21   
22   Edit
23   ----
24   
25   After creating a translation, or to edit an existing translation, open
26   the ``.po`` file. While they can be edited by hand, there are also tools
27   that make working with gettext files easier.
28   
29   Make sure the `.po` file:
30   
31   - Is a valid UTF-8 text file.
32   - Has the header filled out appropriately.
33   - Translates all messages.
34   
35   
36   Verify
37   ------
38   
39   After working on the catalog, verify that it compiles and produces the
40   correct translations.
41   
42   ```
43   $ python setup.py compile_catalog
44   ```
45   
46   Try loading your translations into some sample code to verify they look
47   correct.
48   
49   
50   Submit
51   ------
52   
53   To submit your translation, create a pull request on GitHub.
